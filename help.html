Adress Solver Help

Vous avez des problèmes avec cet outil ?

Cet outil vous permet de faire deux choses :

  • Trouver tous les détails d'une adresse IP, dans laquelle vous pouvez spécifier la taille d'un NET-ID ou non... celui-ci est optionnel, si vous ne le renseignez pas l'algorithme se basera sur l'adresse rentrée pour faire ces calculs, si vous le renseignez il se basera sur l'adresse et le net-ID
  • Si vous êtes ici c'est surement pour celui-ci.
    Le dump d'adress IP permet de traiter simultanément plusieurs adresses IP en même temps les calculs sont similaires avec la fonction précédente. Les adresses IP doivent être séparés par une virgule, les résultats seront affichés en bas de la page.
